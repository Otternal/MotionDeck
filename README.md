## Overview
**MotionDeck** is a Roblox Studio plugin written in **Luau** that is designed to help developers generate procedural idle animations through a slider based interface. It aims to lower the barrier for animation creation by giving users an intuitive alternative to keyframe animation.

# Development Tools
- **Roblox Studio** - UI creation
- **Rojo** - file to studio synchronisation
- **VS Code** - primary development enviornment
- **Selene** - lua linting
- **StyLua** - formatting
- **Luau Language Server** - IntelliSense + type checking
- **Luau LSP Companion Plugin** - realtime instance inference from Roblox Studio
